## 📚 [문헌 연구] JCCR 논문 분석 및 전략 고도화

김선회·윤순진(2025), "IPCC 정책결정자를 위한 요약 보고서의 계량 텍스트 분석" 논문을 통해 도출한 기술적/기획적 인사이트 정리.

### 1. ⚙️ 데이터 엔지니어링: 청킹(Chunking) 전략
- **문제점:** SYR SPM은 텍스트가 매우 압축적(약 378문장)이라 검색 효율이 낮을 수 있음.
- **해결책:** 논문의 분석 단위를 참고하여 **'5개 문장 단위 묶음(5-sentence Chunk)'**을 기본 청킹 단위로 설정.
- **기대효과:** 문맥(Context)을 보존하면서도 통계적 유의성을 확보한 검색 결과 도출 가능.

### 2. 🎭 페르소나 및 프롬프트: IPCC 6차 보고서의 톤앤매너
- **메시지 중심 이동:** '물리적 현상 증명' → **'정책적 실천 및 행동 촉구'**로 답변 톤 조정.
- **용어의 직관성:** - 학술적 용어(`anthropogenic`)보다 직관적이고 책임 소재가 명확한 단어(`human activity`, `human-caused`) 우선 사용 유도.
  - 답변 생성 시 "인류의 활동이 기후 변화의 주된 원인임"을 명료하게 전달하도록 프롬프트 설계.

### 3. 🔑 검색 최적화를 위한 키워드 사전 (Metadata/Keyword)
- **가중치 부여 키워드 (Hot Keywords):**
  - 목표: `1.5°C`, `2°C`, `탄소중립(net-zero)`, `한계초과(overshoot)`
  - 정책: `기후회복적 발전`, `재원(finance)`, `포용적(inclusive)`, `공동편익(co-benefit)`, `SDG`, `NDC`
- **지양/퇴조 키워드 (Cold Keywords):**
  - `SRES`, `안정화(stabilization)`, `Annex B`, `인위적인(anthropogenic)`
- **반영 계획:** RAG 검색 시 위 'Hot Keywords'가 포함된 문서에 가중치를 주는 가중 검색(Weighted Search) 검토.
