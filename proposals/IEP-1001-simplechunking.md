# IEP-1001: 단순청킹을 통한 Context Recall 성능 개선

| 항목 | 내용 |
| :--- | :--- |
| **상태** | `Completed` |
| **작성일** | 2026-03-31 |

## 동기
- **IEP-1000**에서 측정된 Context Recall 성능을 개선하고자 함
- 청크 사이즈와 오버랩 크기를 조절하는 단순 청킹 방식(Simple Chunking)의 효용성 검증
- IPCC 보고서의 서술 구조(설명→데이터→결론) 특성상, 청크 사이즈가 커질수록 문맥 이해도가 높아질 것이라는 가설 검증

## 진행

**질문 유형별 Context Recall**

| 질문 유형 (Type) | 1회차 | 2회차 | 평균 | 비고 |

| **01. 사실 확인** | 0.5952 | 0.5952 | 0.5952 | 기준점 |

| **03. 의견/예측** | 0.8875 | 0.8875 | 0.8875 | 기준점 |
| **04. 범위 밖** | 0.9095 | 0.8595 | 0.8845 | 기준점 |
| **전체 평균** | **0.8205** | **0.8080** | **0.8143** | 기준점 |

## 분석
- **case2**의 가장 큰 청크사이즈를 사용하는 방법이 IPCC 보고서의 문맥 파악에 있어서 가장 효과적이었음
- 특히 **사실 확인** 유형에 있어서  **case2**가 **case1** 보다 **0.3**정도의 큰 성능 향상을 보임
