## 분석
- 청크 사이즈 증가에 따라 전반적으로 Context Recall이 향상되는 경향을 보임
- 특히 **사실 확인** 유형에 있어서 **CASE 3(1000)**이 **CASE 1** 보다 **0.27** 정도의 큰 성능 향상을 보임
- **CASE 4(1200)** 가 전체 케이스 중 Context Recall **0.8567**로 최고 성능을 기록
- 청크 사이즈가 1200을 넘어서면(CASE 5, CASE 6) Context Recall이 다시 하락하는 경향을 보임 → 역전 현상 확인
- **CASE 6(2000)** 은 NaN이 **30개**로 급증하여 데이터 품질 측면에서 가장 불안정함
- 청크 사이즈 증가가 무조건적인 성능 향상을 보장하지 않으며, **약 1000~1200 구간에서 최적점**이 존재하는 것으로 추정됨
