# 🧭 IPCC-Navigator: 기후 과학 데이터 내비게이터

**IPCC-Navigator**는 전 세계에서 가장 공신력 있는 기후 변화 보고서인 **IPCC(기후 변화에 관한 정부 간 협의체) 제6차 평가보고서(AR6)**를 기반으로 합니다. 일반인의 궁금증에 대해 인공지능이 답변하되, 모든 답변의 근거를 실제 보고서 원문과 매칭하여 제공하는 **신뢰 기후 정보 탐색 서비스**입니다.

### 1. Requirements
* **Python:** 3.10+
* **Local LLM:** Ollama (Llama 3)
* **Vector DB:** ChromaDB or FAISS
